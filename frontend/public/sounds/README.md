# Sound assets for ExamAI frontend

Place sound files in this folder with the names used by the app below.

Expected filenames:
- ui-click.mp3
- ui-hover.mp3
- boss-entrance.mp3
- correct-hit.mp3
- wrong-hit.mp3
- victory-fanfare.mp3
- defeat-thud.mp3
- forge-start.mp3
- forge-complete.mp3
- dungeon-open.mp3
- toast-success.mp3
- toast-error.mp3
- toast-xp.mp3

The app loads these files via `/sounds/<name>.mp3`.
